## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="621d3-152">Evitando ataques de scripts entre sites (XSS)</span><span class="sxs-lookup"><span data-stu-id="621d3-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="621d3-153">Sempre que você aceitar a entrada de um usuário e reexibir essa entrada em seu site, você potencialmente abrirá seu site para ataques de scripts de sites cruzados (XSS).</span><span class="sxs-lookup"><span data-stu-id="621d3-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="621d3-154">Em teoria, um hacker mal-intencionado poderia enviar código JavaScript que é executado quando a entrada é reexibida.</span><span class="sxs-lookup"><span data-stu-id="621d3-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="621d3-155">O JavaScript pode ser usado para roubar senhas de usuário ou outras informações confidenciais.</span><span class="sxs-lookup"><span data-stu-id="621d3-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="621d3-156">Normalmente, você pode derrotar ataques XSS codificando qualquer entrada que você recuperar de um usuário antes de exibi-lo em uma página da Web.</span><span class="sxs-lookup"><span data-stu-id="621d3-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="621d3-157">No entanto, a codificação HTML da saída &lt;&gt; do Editor HTML não apenas codificaria tags de script, mas também codificaria todas as tags HTML.</span><span class="sxs-lookup"><span data-stu-id="621d3-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="621d3-158">Em outras palavras, você perderia toda a formatação, como o tipo de fonte, o tamanho da fonte e a cor de fundo.</span><span class="sxs-lookup"><span data-stu-id="621d3-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>
