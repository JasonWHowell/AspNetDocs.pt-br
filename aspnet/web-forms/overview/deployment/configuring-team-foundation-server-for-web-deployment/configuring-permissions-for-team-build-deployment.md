## <a name="configuring-web-server-permissions"></a><span data-ttu-id="a629c-119">Configurando permissões do servidor Web</span><span class="sxs-lookup"><span data-stu-id="a629c-119">Configuring Web Server Permissions</span></span>

<span data-ttu-id="a629c-120">Conforme descrito em [escolhendo a abordagem certa para a implantação na Web](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), há duas abordagens principais que você pode usar se quiser implantar pacotes da Web em um servidor Web remoto:</span><span class="sxs-lookup"><span data-stu-id="a629c-120">As described in [Choosing the Right Approach to Web Deployment](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), there are two main approaches you can use if you want to deploy web packages to a remote web server:</span></span>

<span data-ttu-id="a629c-129">Por outro lado, a abordagem do manipulador de Implantação da Web oferece várias vantagens:</span><span class="sxs-lookup"><span data-stu-id="a629c-129">In contrast, the Web Deploy Handler approach offers various advantages:</span></span>
