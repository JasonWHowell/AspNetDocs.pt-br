# <a name="how-do-i-use-the-aspnet-ajax-resizablecontrol-extender"></a>[Como faço para:] Usar o extensor ResizableControl do ASP.NET AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

O extensor ResizableControl do ASP.NET AJAX Control Toolkit permite que o desenvolvedor especifique uma região da página da Web que o usuário pode redimensionar dinamicamente. Este vídeo demonstra como configurar a aparência da região redimensionável e como executar o script do lado do cliente no redimensionamento.
