# <a name="measuring-the-business-value-of-ajax"></a>Medir o valor comercial do AJAX

por Eric Lee

O ASP.NET AJAX é bem conhecido por sua capacidade de melhorar a experiência do usuário de aplicativos Web. Com o Visual Studio Team System, você também pode medir o valor comercial de adicionar o ASP.NET AJAX aos seus aplicativos Web. Especificamente, você pode usar os recursos de teste de carga e de Web do Visual Studio Team System para explorar como a diferença no tamanho da transmissão de dados HTTP afeta as características de desempenho e escalabilidade do aplicativo Web.
