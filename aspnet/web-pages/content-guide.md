# <a name="webmatrix-content-guide"></a>Guia de conteúdo do WebMatrix

> [!NOTE] 
> O WebMatrix não é mais recomendado como um ambiente de desenvolvimento integrado para ASP.NET Páginas da Web. Use [visual studio](xref:web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) ou visual studio [code](https://code.visualstudio.com/).

WebMatrix é um conjunto de ferramentas que fornece uma maneira fácil de construir sites para microsoft windows. Ele inclui o IIS Express (um servidor web de desenvolvimento), ASP.NET (uma estrutura de programação web) e o SQL Server Compact (um banco de dados incorporado). Para ver um rápido tour pelo WebMatrix, consulte Visão geral do [WebMatrix](https://www.microsoft.com/web/webmatrix/). Para revisar os principais recursos do WebMatrix, consulte [Os Recursos do WebMatrix](https://www.microsoft.com/web/webmatrix/features/).
